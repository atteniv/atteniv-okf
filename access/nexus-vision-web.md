The web app is the free public face of [Nexus Vision](../products/nexus-vision.md).
No credit card and no lengthy onboarding — a focused workspace for exploring
state rules around remote and multi-state employment.

# What It Offers

* **Plain-English summaries** of each state's rules for HR and payroll audiences.
* **Natural-language Q&A** over the corpus, with citations to the exact statute
  sections relied on.
* **Complexity ratings** — a color-coded posture map (no income tax / baseline /
  guardrail / red-flag).
* **Single- and multi-state scenarios** — describe a situation in plain language
  and get one synthesized, cited answer.

# Coverage

All 50 states + DC for profiles and complexity ratings; a subset of states live
for grounded Q&A (20 states at launch), with the general corpus as fallback.

# Note on URL

The canonical user-facing host is `nexus.atteniv.com`. The API and MCP server
are served separately from `nexus.atteniv.app`.
